# 🛠️ YHG Assets Handler (Beta)

**YHG Assets Handler** is a powerful and efficient tool designed to read, analyze, and modify Unity Assets and Bundle files. Equipped with dedicated plugin support for various asset types, it streamlines the process of modding and asset management.

### 🔌 Major Plugin Support
* **Texture2D (28):** High-performance texture exporting (as `.png`) and seamless texture importing.
* **Mesh:** Smooth exporting and importing of 3D Mesh models.
* **AudioClip:** Easily export and import game audio files and sound effects.

---

### 📚 Credits & Libraries Used
This project wouldn't be possible without these amazing open-source libraries, frameworks, and tools:

* **[AssetsTools.NET](https://github.com/nesrak1/AssetsTools.NET):** For handling Unity assets and bundle files efficiently.
* **[AssetRipper](https://github.com/AssetRipper/AssetRipper):** Core libraries utilized for advanced texture and deep asset extraction.
* **[Assimp (Open Asset Import Library)](https://github.com/assimp/assimp):** For providing a powerful, unified interface to import and process various 3D model formats.
* **[SixLabors.ImageSharp](https://github.com/SixLabors/ImageSharp):** For high-performance, cross-platform image processing and texture exporting.
* **[FMOD](https://www.fmod.com/):** For powering the audio decoding and high-fidelity `AudioClip` processing system.
* **[AssetStudio](https://github.com/Perfare/AssetStudio):** For core Unity asset inspection logic and visualization.
